## 🎯 Overview

A modern, responsive portfolio website built with **Next.js 15**, featuring dark luxury aesthetics and smooth animations. This project serves as a showcase of my journey at **Army Institute of Technology (AIT), Pune**, and my work in hardware-software integration.

**🌐 Live Demo:** [karishma.vercel.app](https://karishma.vercel.app) (Coming Soon!)

## ✨ Features

- 🎨 **Dark Luxury Theme:** Ultra-dark design with purple accents and glass effects.
- ⚡ **Performance:** Built with Next.js 15 for lightning-fast loading.
- 📱 **Responsive:** Fully optimized for mobile and desktop viewing.
- 🛠️ **IoT Focused:** Highlighting real-world engineering projects like Smart Irrigation.

## 🛠️ Tech Stack

- **Languages:** C++, Python, TypeScript
- **Frameworks:** Next.js 15, React 19, Tailwind CSS
- **Hardware:** Arduino, Sensors (Soil Moisture, pH)
- **Design Tools:** Figma, Canva

## 📁 Key Projects

- **Smart Irrigation System:** An automated IoT prototype for precision farming.
- **Antigravity Project:** Collaborative physics experimentation and structural design.
- **Next Niche Branding:** Full visual identity and flyer design for campus events.
